# **AI Children’s Storybook App – Specification Brief**

**Project Goal:**
Build a web-based, AI-powered children’s storybook platform using Laravel + Inertia.js (Vue.js) where users can generate illustrated, narrated, and optionally video-animated storybooks automatically from topics, lessons, and selected styles.

---

## **1️⃣ User Flow**

**Step 1: Story Setup**

* User selects a topic or enters a custom one.
* User selects story parameters:

  * Lesson type (moral, educational, language, etc.)
  * Age group
  * Story length (# of pages)
  * Illustration style (cartoon, watercolor, 3D, etc.)

**Step 2: Content Generation**

* **AI Storyline**: Generate page-by-page text using GPT-5.4.
* **Optional Quiz**: Generate 1–3 questions per page based on the story.
* **AI Image Generation**: Generate images per page using `gpt-image-1`.
* **AI Narration**: Generate page narration using OpenAI `gpt-voice`.
* **Optional Video**: Animate the image + narration per page using Runway Gen-2 or Kaiber.

**Step 3: Preview & Edit**

* Users can preview each page: text + image + audio/video.
* Edit text or re-generate image/audio if needed.

**Step 4: Export / Share**

* Download PDF of storybook.
* Export Turn.js interactive storybook.
* Optional: Export video compilation of full story.

---

## **2️⃣ Backend Architecture (Laravel)**

**Models:**

* `User` – manages accounts, subscription/credits.
* `StoryProject` – stores project metadata: title, topic, style, lesson type, status.
* `StoryPage` – page_number, text_content, image_url, audio_url, video_url, quiz_questions.
* `AIJob` – tracks async tasks (text, image, audio, video).
* `FeatureLevel` – basic/pro/video/quiz settings.

**Features:**

* Async job queue for AI generation (text → image → audio → video).
* Credits/limits per user for feature levels.
* Media storage: S3 or equivalent cloud storage.

---

## **3️⃣ Frontend (Vue + Inertia)**

**Pages / Components:**

1. **Dashboard:** List all story projects, generation progress, and status.
2. **Story Editor / Generator:** Guided multi-step page for topic → AI generation → edit/preview.
3. **Story Viewer:** Turn.js interactive page-flipping interface with:

   * Image + Text + Audio
   * Video (optional)
   * Quiz (optional)
4. **Export / Share:** PDF, HTML, or video download options.

## **1️⃣ Text / Storyline Generation**

* **Model:** `gpt-5.4` (or `gpt-4.1` if you want a balance of cost & speed)
* **Strength:** best for creative storytelling, age-appropriate content, lesson-based stories, and quiz generation.
* **Usage:** Generate the story page by page, plus any quiz questions or lesson notes.

---

## **2️⃣ Image Generation**

* **Model:** `gpt-image-1` (OpenAI’s current image generation model)
* **Strength:** high-quality, controllable style (cartoon, watercolor, 3D), safe-for-children output.
* **Advantage:** Unified with OpenAI ecosystem; no need for DALL·E 3 or external API.
* **Usage:** Generate illustrations for each story page.

---

## **3️⃣ Audio / Narration**

* **Model:** OpenAI `gpt-voice` or `whisper` + TTS

  * Whisper is for audio transcription, but OpenAI now supports text-to-speech (`gpt-voice`) capable of natural narration.
* **Strength:** realistic, expressive voices suitable for children.
* **Usage:** Generate narration for each page automatically or let user choose voice/language.

---

## **4️⃣ Video Generation**

* Latest AI video models:

  * **Runway Gen-2** or **Kaiber AI** — can animate static images and combine with narration.
  * **OpenAI**: Video generation is emerging but not yet fully public for high-quality long videos; Runway/Kaiber are more production-ready today.
* **Usage:** Take the generated image + narration → full-page short video for storybook page.

---

## **Suggested Modern Pipeline**

1. **User input:** topic + style + lesson type
2. **Text generation:** GPT-5.4 generates story & optional quizzes
3. **Image generation:** `gpt-image-1` creates illustrations
4. **Audio:** `gpt-voice` narrates story text
5. **Optional Video:** Runway/Kaiber creates full-page animation with image + narration
6. **Frontend:** Turn.js displays pages (image + audio/video + text)
